# Yejin Kwon

운영 가능한 서비스를 만드는 **시스템 엔지니어** · **데이터 엔지니어** 지향 개발자입니다.  
**Terraform 기반 IaC**와 **GitHub Actions 기반 CI/CD**로 인프라·배포를 자동화하고, **로그/패킷 기반 트러블슈팅**과 **Runbook 문서화**로 신뢰성을 높입니다.  
또한 **Python 기반 데이터 처리**과 **ML/CV 기능**을 **API로 서비스화**하여, 데이터 기능을 운영 관점에서 안정적으로 제공하는 **엔드투엔드 흐름**을 지향합니다.
